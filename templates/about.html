Research on Sentiment Analysis

This project focuses on developing a Sentiment Analysis System to predict whether they are positive, negative or neutral. The system leverages machine learning techniques, specifically Logistic Regression, to analyze tweets and categorize them into positive, negative, or neutral sentiments.

Key objectives included achieving high accuracy in sentiment classification, real-time analysis capabilities, and scalability. The system was developed using Python, Flask for deployment, and the it utilized Logistic regression as a machine learning algorithm. Preprocessing techniques such as tokenization, normalization, and handling of multilingual text were implemented to improve accuracy. The final model achieved an accuracy of 92%, closely approaching the target of 95%.
